1. Greetable (единый "интерфейс" для взаимодействия сервера с обработчиками) - api
2. Обработчики для Greetable - handlers
3. Сервер - server

Обработчики потребляют интерфейс Greetable.
Сервер работает с объектами классов унаследованных от Greetable, и опосредовано, через classpath с Greetable.

4. Клиент - client

Клиент обращается к серверу, сервер отвечает клиенту
