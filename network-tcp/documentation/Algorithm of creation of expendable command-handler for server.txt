1. Создаём абстрактный класс для сервера, через который будем обращаться к объектам классов-комманд
2. Создаём реализации-наследники этого абстрактного класса
3. Используя reflection загружаем объекты из конфигурационного файла и кладём их в Map
4. Учим сервер доставать объект по имени из Map и вызывать у него метод
